# Movie_Recommender
Collaborative filtering and Content-based filtering recommenders.

Collaborative filtering:
  * Ratings: https://media.geeksforgeeks.org/wp-content/uploads/file.tsv
  * Movie titles: https://media.geeksforgeeks.org/wp-content/uploads/Movie_Id_Titles.csv

Content-based filtering:
  MovieLens-100K Kaggle dataset https://www.kaggle.com/datasets/rajmehra03/movielens100k?select=tags.csv
  * movies.csv  contains: movieId, title, genres (pipe-separated).
  * ratings.csv  contains: userId, movieId, rating, timestamp.
  * Short descriptions: tags.csv  contains: userId, movieId, tag, timestamp.
  * links.csv  contains: movieId, imdbId, tmdbId. See www.imdb.com and https://www.themoviedb.org/.
